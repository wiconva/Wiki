Echo y cls.

El comando 'echo' evita que se imprima todo en el promt del cmd y el comando cls borra todo del promt.

Comandos para la adminsitración ISS.

Los comando para ISS se ejecutan mediante la aplicación que tiene integrada appCmd + el comando. Para iniciar una aplicación dado el nombre se usa : "%windir%\system32\inetsrv\appcmd start site /site.name:AppStudio".
Se puede utilizar los comandos start, stop, restart, delete, list...

Crear directorio si no existe.

Basta con utilizar el comando if not exists la ruta de evaluación y luego el comando mdir con la ruta de creación.

Comprimir archivos en zip.

Se realiza un cambio a la ubicación donde se encuentra los archivos a comprimir y se ejecuta tar, pasando como parámetros el nombre del paquete generado y el nombre de la carpteta base.

Copiar archivos.

Cambiar la ubicacion a donde se encuentra el archivo a copiar y luego pasar como parámetro el origen y el desino del archivo copiado, se pueden cambiar las extensiones como en el ejemplo.

Extraer ZIP.

Cambiar la ubicacion a donde extraerá el archivo y ejecutar el comando tar -xf pasando como parámetro la ruta del paquete.

Borrar archivos.

Cambiar la ubicacion a donde borrará el archivo y ejecutar el comando "Del", pasando como parámetro el nombre del archivo a borrar.

Crear tarea programada.

Para crear una tarea en el programador de tareas de Windows se utiliza el comando schtask y se pasan como parámetros el tipo de ejecución, el nombre de la tarea creada, la acción de la tarea, si se desea que se ejecute cuando el usuario sea otro y tenga permisos de administrado se pasa /ru System. Si la tarea ya existe no falla la ejecución del comando.

Ejecutar tarea programada.

Para ejecutar la tarea programada se utiliza el comando schtask y se pasa como argumento /run y el nombre de la tarea .

Finaliza tarea programada.

Para finalizar una tarea programada que esta en ejecucion se utiliza el comando schtask y se pasa como argumento /end y el nombre de la tarea .

Minimizar todas las ventanas.

Para minimizar todas la ventanas se ejecuta un comando en powershell desde la consola CMD con powershell -command.
