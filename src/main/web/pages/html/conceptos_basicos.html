Conceptos Básicos

Es un standar compuesto por estas tecnologías:

Estructura:

Es la estructura básica de un documento HTML 5.

Etiqueta title:

Va dentro de la etiqueta meta, su función es cambiar el titulo de la página y ayuda con el posisionamiento al momento de la busqueda dentro un navegador.

Etiqueta link:

Va dentro de la etiqueta meta, su función es enlazar las hojas de estilos o los archivos Java Script que le daran utilizados para el dinamismo de la página.

Estructura de una Página HTML 5:

Esta es la estructura que ha ido evolucionando, inicialmente se realizaban con tablas para el Layout, luego con ayudas del Div llegando a la estructura que se muestra.

Insertar una Imagen:

si se deasea agregar un pie a la imagen se utiliza la etiqueta figcaption dentro de figure.
