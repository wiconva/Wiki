Conceptos Básicos

Es un estandar para dotar de apariencia una páginca WEB, Se puede incluir através de una hoja con extensión .css, en el header o directamente sobre una etiqueta.

Estructura:

Consta de un selector seguido de la propiedades a manipular

Tipos de selector:

Los selectores pueden ser de tipo etiqueta: Nombre de la etiqueta, id: # o clase:'.'.

Etiqueta display y float:

Diplay puede ajustar el comportamiento del contenedor y pasarlo de Block a inline, esto hace que no ocupe el ancho total de la página y quita el salto de línea. El atributo float permite ubicar libremente el contendor y que este se ajuste al tamaño del contenido, tambien sacandolo del flujo de control de los elemento y ubicando el siguiente al lado, para romper este comportamiento el siguiente elemento debe llevar la etiqueta clear: both.

Clear:

Lleva al flujo normal al elemento que lo precede si se ha aplicado la etiqueta float y no se ubica al su lado como se esperaria, agregando nuevamente el comportamiento de block a la etiqueta.

Margin y padding:

El atributo Margin ajusta el margen desde el contenedor hasta el siguiente elemento y padding ajusta la distancia entre los elementos que tiene el contenedor y el borde del mismo. Se pueden indicar cuatro valores uno para cada ubicación cardinal; dos : vertical y horizontal o uno para que sea igual en los cuatro puntos.

Border y border-radius:

Se utiliza para dar colocar el borde al contenedor y con radius para colocar el radio en las esquinas donde se encuentran los bordes.
